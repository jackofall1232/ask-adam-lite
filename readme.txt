== Description ==
Shortcode: [ask_adam_lite]
Widget: Ask Adam Lite (Chat Widget)

== Installation ==
1. Upload the ZIP via Plugins → Add New → Upload Plugin.
2. Activate.
3. Go to Ask Adam Lite → Settings, add your OpenAI key.
