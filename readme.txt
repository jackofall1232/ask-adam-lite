Lite version of Ask Adam — a secure, minimal AI chat widget and knowledge base for WordPress.
Easily drop it into your site with a shortcode or floating widget. Upgrade to Pro for
multi-provider support, real-time search, and advanced customization.

== Description ==

Shortcode: [ask_adam_lite]  
Widget: Ask Adam Lite (Chat Widget)

== Installation ==

1. Upload the ZIP via Plugins → Add New → Upload Plugin.
2. Activate.
3. Go to Ask Adam Lite → Settings and add your OpenAI API key.

== Frequently Asked Questions ==

= Do I need an OpenAI account? =
Yes. Ask Adam Lite requires your own OpenAI API key.

= How do I upgrade to Pro? =
A Pro version with more features is available at https://www.askadamit.com
